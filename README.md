# REST API for a shoe store

## Description

created using [Nest](https://github.com/nestjs/nest) framework and [Prisma](https://www.prisma.io/) ORM

Features include:
- CRUD operations on the product
- Authentication
- (more to be added...)

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
